# Трекер тренировок и привычек

PWA для iPhone: тренировки (подходы×повторы + кардио), привычки со стриками, вес,
счётчики (75-дневный челлендж и дни трезвости). Данные хранятся в самом телефоне,
работает офлайн, бэкенд не нужен.

## Локальный запуск

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Тесты

```bash
npm test
```

## Деплой на GitHub Pages

1. Создать репозиторий на GitHub и запушить эту папку.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
3. Открыть выданный URL на iPhone в Safari → «Поделиться» → «На экран Домой».

Теперь иконка на домашнем экране открывает приложение в полноэкранном режиме, работает офлайн.

## Бэкап

Данные лежат в браузере, поэтому раз в неделю заходи в **Настройки → Экспорт** и
сохраняй файл (например, в iCloud). Восстановление — **Настройки → Импорт**.

## Синхронизация с сервером (опционально)

Можно хранить данные в облаке (на своём сервере) и синхронизировать между устройствами.
Бэкенд — лёгкий Node+SQLite-сервис в папке [`server/`](server/README.md), без внешних зависимостей.

1. Развернуть `server/` по инструкции [server/README.md](server/README.md) (без поддомена — через путь `/trackerapi/`).
2. В приложении: **Настройки → Синхронизация** → ввести адрес API и ключ.
3. Дальше изменения сами уходят на сервер; localStorage остаётся офлайн-кэшем.
   Очистил Safari или сменил телефон — после ввода ключа данные подтянутся обратно.

## Структура

```
index.html        разметка + таб-бар
styles.css        тема (Duolingo-в-стиле-iOS)
src/logic.js      чистая логика (даты, счётчики, стрики, агрегация, стартовые данные)
src/storage.js    localStorage + экспорт/импорт
src/charts.js     SVG-графики
src/app.js        рендер экранов и обработчики
manifest.json     PWA-манифест
sw.js             service worker (офлайн-кэш)
icons/            иконки приложения
test/             тесты (node --test)
docs/superpowers/ спека и план реализации
```
