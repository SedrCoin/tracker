# Трекер тренировок и привычек — дизайн (PWA)

**Дата:** 2026-06-25
**Статус:** утверждён, готов к реализации

## Цель

Заменить ручное заполнение заметки в телефоне на удобную мини-аппку для iPhone, в которую владелец каждый день быстро записывает тренировки и привычки, и которая сама считает счётчики (челлендж, дни трезвости) и строит графики прогресса.

## Технический подход

- **Чистый HTML + CSS + JavaScript, без сборки** (без npm/React/бандлеров). Файлы кладутся на GitHub Pages как есть.
- **PWA**: `manifest.json` + service worker → работает офлайн, добавляется на домашний экран iPhone, выглядит как нативная аппка.
- **Данные**: `localStorage` (объёмы маленькие, этого достаточно). Без бэкенда и базы данных.
- **Графики**: рисуются вручную на SVG, без внешних библиотек (чтобы всё работало офлайн).
- **Хостинг**: GitHub Pages (бесплатно, HTTPS). Аналоги: Netlify/Vercel/Cloudflare Pages.

## Структура файлов

```
index.html        — разметка, контейнеры экранов
styles.css        — стили, тема Duolingo-в-стиле-iOS
app.js            — вся логика (состояние, рендер, обработчики)
storage.js        — загрузка/сохранение/экспорт/импорт состояния + сидинг
charts.js         — построение SVG-графиков
manifest.json     — PWA-манифест
sw.js             — service worker (офлайн-кэш)
icons/            — иконки приложения (PNG разных размеров)
```

(Допустимо объединить JS в один файл, если так чище. Главное — модули логически разделены.)

## Экраны (нижний таб-бар)

### 1. Сегодня (главный)
- **Счётчики сверху** — компактные цветные карточки:
  - Челлендж: «Осталось N дней» + мелким «День X из 75».
  - Без алкоголя: «N дней».
  - Без спреев: «N дней».
- **Взвешивание** — плашка: текущий/последний вес, дата следующего взвешивания. В день взвешивания — подсветка «Сегодня день взвешивания!» и поле ввода веса.
- **Тренировка**:
  - Кнопки-пресеты: «Турник», «Брусья». Нажатие → ввод подходов (числа), подходы можно добавлять/убирать.
  - «＋ своё» — добавить любое упражнение. Тип `reps` (подходы×повторы) или `cardio` (поле «дистанция/время» текстом).
- **Привычки** — чек-лист галочек на текущий день, у каждой виден стрик «N дней подряд».
- **Заметка** на день — свободный текст.
- **Навигация по дням**: стрелки ‹ › для переключения на прошлые/будущие дни (чтобы дозаписывать пропущенное).

### 2. Статистика
- Линия/столбцы суммы повторов по дням — отдельно по каждому упражнению (Турник, Брусья, …).
- Всего тренировок.
- **Вес по времени** — линейный график замеров.
- Календарь-сетка привычек со стриками.

### 3. Настройки
- Даты счётчиков (старт «без алко», «без спреев»; якорь и остаток челленджа).
- Взвешивание: интервал (14 дней) + якорная дата.
- Список упражнений-пресетов (добавить/удалить/переименовать).
- Список привычек (добавить/удалить/переименовать).
- **Экспорт** (скачать JSON-бэкап) / **Импорт** (восстановить из файла).

## Логика счётчиков

- **Челлендж — обратный отсчёт.** Якорь: 2026-06-22 = осталось 75. `remaining(today) = 75 - (today − 2026-06-22 в днях)`. Финиш (0) = 2026-09-05. На 2026-06-25 → осталось 72.
- **Без алкоголя:** дни с 2025-09-27.
- **Без спреев:** дни с 2026-05-02.
- У трезвых счётчиков — кнопка **«Сброс»** (обнуляет старт на сегодня) с подтверждением.

## Взвешивание

- Интервал 14 дней, якорь — старт челленджа 2026-06-22.
- График замеров: 22.06 → 06.07 → 20.07 → 03.08 → 17.08 → 31.08 → …
- Хранится список `{ дата, вес }`. Известный замер: **78.8 кг на 2026-06-22** (заметка владельца упоминала «85 день» — цифры не сходятся, дата правится в один тап).
- В день взвешивания UI подсвечивает и предлагает ввести вес. Показывает дату следующего.

## Привычки

- Стартовый набор: Французский, Шахматы, 10 отжиманий, Медитация.
- Галочка вкл/выкл на каждый день; стрик = число последовательных выполненных дней до сегодня.
- Список редактируется в Настройках.

## Модель данных (localStorage, один JSON)

```js
{
  settings: {
    noAlcoholStart: "2025-09-27",
    noSpraysStart: "2026-05-02",
    challenge: { anchorDate: "2026-06-22", remainingAtAnchor: 75 },
    weighIn:   { anchorDate: "2026-06-22", intervalDays: 14 }
  },
  exercises: [                         // пресеты + добавленные
    { id, name: "Турник", type: "reps", preset: true },
    { id, name: "Брусья", type: "reps", preset: true }
  ],
  habits: [
    { id, name: "Французский" }, { id, name: "Шахматы" },
    { id, name: "10 отжиманий" }, { id, name: "Медитация" }
  ],
  weighIns: [ { date: "2026-06-22", weight: 78.8 } ],
  days: {
    "2026-06-21": {
      workouts: [
        { exerciseId, name: "Турник", type: "reps", sets: [6,4,4,2,4] },
        { exerciseId, name: "Брусья", type: "reps", sets: [5,4,4,5,6] }
      ],
      habits: { "<habitId>": true|false, ... },
      note: ""
    }
    // ... остальные дни
  }
}
```

Для `cardio`-упражнений вместо `sets` — поле `value` (текст «5 км / 30 мин») и опционально `note`.

## Визуал (Duolingo-в-стиле-iOS)

- Яркая сочная палитра: фирменный зелёный + акценты (синий/оранжевый/фиолетовый), каждый счётчик своим цветом.
- Крупные скруглённые карточки; «пухлые» 3D-кнопки с нижней тенью (как в Duolingo); крупный дружелюбный шрифт.
- Кольца/полоски прогресса, стрики с огоньком 🔥, лёгкое празднование (конфетти) при закрытии всех привычек дня.
- Аккуратная iOS-раскладка: нижний таб-бар, безопасные отступы (safe-area), плавные нажатия, отзывчивые тапы.
- Тёмный/светлый — по системной теме (можно начать со светлой яркой).

## Бэкап и сохранность данных

- Данные живут в браузере → риск потери при очистке Safari.
- **Экспорт**: скачивание JSON-файла со всем состоянием (рекомендация — раз в неделю в iCloud).
- **Импорт**: восстановление из ранее сохранённого файла (с подтверждением перезаписи).

## Вне рамок первой версии (YAGNI)

- Облачная синхронизация между устройствами.
- Аккаунты/авторизация.
- Пуш-уведомления и напоминания.
- Соцфункции.

## Критерии готовности

1. Открывается с домашнего экрана iPhone, работает офлайн.
2. Можно записать тренировку (пресет и своё), отметить привычки, ввести вес, написать заметку — за сегодня и за прошлые дни.
3. Все три счётчика и взвешивание считаются корректно (проверка на известных датах: челлендж = 72 на 25.06).
4. Статистика показывает графики повторов, веса и календарь привычек.
5. Экспорт/Импорт сохраняет и восстанавливает всё состояние.
6. Стартовые данные (21–24.06 + вес) предзаполнены.
